Add images

Add images you can later in emails.

To add an image to SuperOffice Marketing:

  1. Select the Images tab.

  2. Optional: Select a folder or create a new one.

    How do I create a mailing folder

  3. Click the Add button. This takes you to the Image properties screen.

  4. Enter a descriptive name for the image in the Name field.

  5. Click Upload files (or drag here), select the image to add, and click Open. or Drag an image from your PC and drop it on Upload files (or drag here). The preview displays the image.

  6. Click OK when you are done.

Tip

When editing a message, you can also add images from the royalty-free image library.
