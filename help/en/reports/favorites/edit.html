Edit favourite reports

Note

This assumes that you have added some favorite reports.

To edit a favourite report:

  1. Click the Reports button in the Navigator to open the Reports screen.

  2. Select the Favourites tab. The favorite reports are displayed in the report list.
  3. Double-click the report you want to edit, or select it and click Edit. A dialog opens in which you can edit the report's details.
  4. Edit the name of the report in the Please enter a name for your report field, as required.
  5. Edit the text in the Report description field, as required.
  6. Click OK.
Note

The report is amended immediately and you cannot undo your changes.
