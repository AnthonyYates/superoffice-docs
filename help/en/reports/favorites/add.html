Add favorite reports

Reports that you use a lot or have edited you can add to the list of favorite reports.

To do this:

  1. Click the Reports button in the Navigator to open the Reports screen.

  2. Select the tab you require from Company, Project, Sale, Activity or Selection.
  3. Select a report from the report list and make the required changes. (See Edit search criteria in reports).
  4. Click Add to favorites at the bottom right of the criteria list. The Add favourite dialog opens.
  5. Enter the name of the report in the Please enter a name for your report field.
  6. Type in the required text in the Report description field. If you have amended any criteria in the report, change the description accordingly.
  7. Click OK. The report is added to the list of favorites.
Tip

You can edit the name and description of a favorite report by double-clicking it or selecting it and clicking Edit. See Edit favorite reports.
