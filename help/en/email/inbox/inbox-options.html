The Inbox

The inbox contains the following options:

Toolbar at the top

Icon Description
New Click here to create a new message. See Compose an email.
Delete
Search Enter text in this field to start searching in the open folder.
INBOX Click here to select a folder. The default folder is Inbox.
By default the messages are sorted by date, with the newest messages at the top of the list. You can also sort by sender and subject in descending or ascending order.

Toolbar at the bottom

Icon Description
Filter: Click here to filter the messages in the inbox.
Refresh: Click here to refresh the inbox.
