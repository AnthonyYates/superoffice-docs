Archive options

These options appear in the Archive options pane of the **SuperOffice CRM options dialog (see SuperOffice CRM options dialog). You can override the default options by using the buttons in your mail client (see Buttons on the mail client toolbar and Buttons in the email window).

Archive email by default when:

For each option you check here, Archive is ON will be set by default for the situation the option refers to. If you uncheck an option, Archive is OFF will be set by default.

created from SuperOffice CRM:

Checking this box enables default archiving of emails that you compose using SuperOffice CRM (see Creating emails from SuperOffice CRM). When you send an email, it will be archived in SuperOffice CRM by default.

written to associates from SuperOffice CRM:

Check this box if you want to archive all outgoing emails written to associates. If you leave this box unchecked, emails sent to people that are registered as associates in SuperOffice CRM will not be archived by default. This option only applies to emails that you compose using SuperOffice CRM (see Creating emails from SuperOffice CRM). Please note that this option is only available if the created from SuperOffice CRM box is checked

created from email client:

Checking this box enables default archiving of emails you compose in your mail client by clicking the New email button in the main window of the mail client. When you send an email, it will be archived in SuperOffice CRM by default. Please note that if you select Archive is OFF in the email window of the mail client, the email will not be archived to SuperOffice CRM (see Turning default archiving on/off).

replying to emails from SuperOffice CRM:

Check this box if you want to enable default archiving of emails you reply to in SuperOffice CRM.

forwarding emails from SuperOffice CRM:

Check this box if you want to enable default archiving of emails you forward in SuperOffice CRM.

sending appointments from SuperOffice CRM:

Check this box if you want to archive by default appointments sent as emails from SuperOffice CRM. You can send an appointment as an email by clicking Task > Send as email in the Appointment dialog. A new email window containing the appointment will then open in your mail client.

sending documents from SuperOffice CRM:

Check this box if you want to archive documents sent as emails from SuperOffice CRM. You can send a document as an email by right-clicking it in the Activities section tab in SuperOffice CRM and selecting Send as email or by clicking Task > Send as email in the Document dialog. A new email window containing the document will then open in your mail client.

Default template when archiving email:

In these two dropdowns you may select the default incoming and outgoing email template.
