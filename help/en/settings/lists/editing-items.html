Edit items

You can change the name or description of an item previously set up in Settings and maintenance.

Note

The name of the item will be changed wherever it is already in use in SuperOffice CRM. If you want to avoid this, delete the old item instead and create a new one (see Delete items).

To edit the name and/or description of an item:

  1. Click the Lists button in the Navigator ().

  2. Click the arrow ( ![icon][img1] ) and select [img1]: ../../media/icons/admin/pil-ned.bmp the required list.

    All previously defined items for the selected list are displayed in the Items list. If you select a new user-defined list, the list will be empty.

  3. Double-click the required item in the Items list. The Edit list item dialog opens.

    Do you want to edit an item in a specialised list?

    Some list types are more specialised than those referred to in this procedure. For more information, see Specialised lists.

  4. Type a new name for the item in the Name field.

    Note
    Note

    This is a mandatory field and must therefore be filled in.

  5. Edit the description of the item, as required, in the Description field.

  6. Click Save to save the changes, or Cancel to reject them.
