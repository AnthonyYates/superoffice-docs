Change labels

You can change labels in all available languages in SuperOffice.

Note

These changes apply only to SuperOffice CRM and not, for example, to reports or labels in Reporter Studio.

To change labels in SuperOffice CRM:

Click the Options button in the Navigator ().

  1. Select the Labels tab.

  2. Check Enable label substitution.

  3. Type in all or part of the label you want to change in the field below the language field on the left. This is the language that has been selected for your user.

    Tip

    If you want to search for labels in another language, select the language in the list on the right and enter the search word in the field below.

  4. Press ENTER. The list will be updated with labels containing the search text.

    Tip

    Hold the mouse pointer over a row to display the resource ID for the label.

  5. Double-click the label you want to change.

  6. Edit the text for the language in question in the Edit field label dialog.

    Note

    It is possible to edit the label for all available languages in this dialog. Not only the selected target language.

  7. Click Save.
