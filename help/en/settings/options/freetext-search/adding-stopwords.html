Add stopwords

To add a stopword manually:

Note

Instead of entering stopwords manually, you can let the system suggest stopwords for you. See Suggest stopwords.

  1. Click the Options button in the Navigator ().

  2. Select the Free text search tab.

  1. Check that freetext search is enabled for SuperOffice CRM (see Enable freetext search).

  2. Click the Add button below the stopword list.

  3. In the New Stopwords dialog, type in one or more stopwords you want to include. As a separator, use any non-alphabetical character, except a full stop.

  4. Click Save.

  5. A message informs you that changes to the stopword list do not have a retroactive effect. In other words, new stopwords will not apply for text entered before the stopword was specified. Click OK.

    Tip

    If you want your changes to have a retroactive effect, you need to regenerate the freetext index. This should be done after you have added stopwords, after the system has suggested stopwords or after importing a large quantity of text. See Regenerate a freetext index.
