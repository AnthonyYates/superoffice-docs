Search options

SuperOffice CRM contains several different search tools:

  • If you click the Find button on the top bar, you open the Find screen, where you can search for data in SuperOffice.
  • You can use FastSearcher or the history list to search for companies, sales, projects and selections.
  • You can use freetext search to search for any text which users have entered in SuperOffice CRM
