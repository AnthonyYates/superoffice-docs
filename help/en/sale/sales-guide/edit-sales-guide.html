Edit a sales guide

This task is done in Settings and maintenance and you need to be an administrator.

To edit an existing sales guide:

  1. Click the Workflow button in the Navigator ().
  2. Select the Sales guide tab.
  3. Select the required sale type in the Sale type list box.
  4. Edit the information about the sale type and its stages by clicking Edit. See Add items to the Sale - Stage list.
  5. Edit the individual stages by following the procedure under Add items in the Sale - Stage list.
  6. Edit follow-ups and documents by double-clicking them in the lists at the bottom. See Add a follow-up for a stage and Add a document for a stage.

The changes are saved automatically.
