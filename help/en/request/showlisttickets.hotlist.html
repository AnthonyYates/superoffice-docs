Add requests to the favourites list

In SuperOffice Service there is a favourites list, to which you can add requests you want to remind yourself to follow up.

Tip

You can also add contacts, companies, selections and mailings to your favourites. These favourites can be accessed by clicking the relevant icon in the Navigator and clicking . See Favourites.

Add requests to the favourites list

How can I add multiple requests to the favourites list?

  1. Search for the relevant requests.
  2. Click the checkbox next to each request you want to add to the favourites list.
  3. Click in the lower left corner of the list.
  4. Select Batch management > Add to my favourites.

Add requests to the faviourites list:

  1. Open the request. (See Process requests.)
  2. Click the favourites icon ( ) in the request header. The star turns yellow , and the request is instantly added to favourites. You do not have to click Save.
Tip

Before transferring responsibility for the request to someone else (see Ask another request handler for help), it is a good idea to add it to your favourites list, to remind yourself to follow it up. It will no longer be displayed under Own open requests when someone else is defined in the Owner field.

View the favourites list

To view the favourites list, do one of the following:

  • Select Requests in the Navigator, and click .
  • Select Requests in the Navigator, and select My favourites.
  • Go to Dashboard, click the list button ( ) next to the title of a panel, and select My favourites.
