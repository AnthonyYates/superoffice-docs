View search results for requests

Once you have got a list of requests using one of the methods described under Find requests, you will be in a screen containing the following sections:

  • Result: This contains all the requests that match your search criteria.
  • Preview: Displays the contents of the request selected under Find requests.

To edit the search criteria, click Show criteria. See Search in SuperOffice Service.

Result

Contains a list of requests that match the criteria specified above. Here you can:

  • Sort the requests by clicking the appropriate column heading.
  • Select a row to display the content of that request in the preview section.
  • Open a request by double-clicking the relevant row. See List requests.
  • Click the menu button in the top right corner to access the change the view options.
Tip

Requests with a red envelope icon ( ) in the Status column are unread, while those with a blue icon ( ) have been read. A yellow envelope icon ( ) means read by the owner, but subsequently amended.

Under the list, you can click the menu button to access the following options:

  • Export: Click here to export the list under Result to a file. You can choose from these formats: Excel, HTML and plain text.
  • Batch management: If you select one or more requests in the list, you can click this button and select an action to perform on all the selected requests simultaneously. For example, you can choose to add the same message to multiple requests, forward multiple requests or delete several requests in a single operation.

Preview

This section of the screen displays the contents of the request selected under Result. Here you have the same options as described under Process requests.

Where is the preview?

If the preview section is not visible, click the menu button in the top right corner and select Show reading pane to the right or Show reading pane below.
