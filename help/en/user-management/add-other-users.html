Add other users

Note

Anonymous users and system users require the SuperOffice Expander Services system license. See Licenses.

To add anonymous users or system users:

  1. Click the Users button in the Navigator ().

  2. Select the Other users tab. The list displays existing anonymous users and system users.

  3. Click Add below the list.

  4. Enter an ID in the User ID field.

  5. Enter, if required, a password for the user in the Password field.

  6. Enter a description of the user, as appropriate, in the Description text box.

  7. Select one of the following options for Type:

    • System user: System users can log in to the system via third-party applications and have all rights.
    • Anonymous: Anonymous users have rights based on the role for anonymous users (see [Edit rights for anonymous users][1]). It is not usually necessary to have more than one anonymous user set up. [1]: ../role/edit-rights-for-anonymous-users.md

  8. Select Active to the right of Login to activate the user.

  9. Click Save when you have finished. The user is added to the list.
