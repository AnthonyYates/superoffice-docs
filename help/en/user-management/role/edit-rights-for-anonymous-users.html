Edit rights for anonymous users

Note

Anonymous users and external users require a separate license. See Licenses.

There is only one role for anonymous users and only the data rights for this role can be edited.

To do this:

  1. Click the Roles button in the Navigator ().

  2. Select the Anonymous tab.

  3. Enter the data rights in the Data rights tab as described under Set data rights for a role.

    What do the different rights mean? [!include[Table showing rights per role](includes/table-role-rights.md)]

The changes are saved automatically.
