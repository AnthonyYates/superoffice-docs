View companies

You can get to the Company screen as follows:

  • You can double-click a row under Find companies in the search results screen.
  • You can click the company's name in screens where this information is displayed, such as the Request screen.

This screen comprises the following sections:

  • At the top of the screen, the company header contains basic information about the contact. See Company header.
  • The Actions button gives access to tasks associated with contacts. See Task buttons for companies.
  • The Requests tab contains the requests associated with this company. Open a request by double-clicking it. See Process requests.
  • The Contacts tab contains the contacts associated with this company. Open a contact by double-clicking the appropriate row. See View contacts.
  • The Info tab contains an overview of the data registered for the active company. The contents of the fields are described in detail under Create companies.

Company header

The header of the Company screen contains basic information about the company: company name, category and priority. In addition, you can click to set the company as a favourite ( ). The company will then be accessible via the favourites list ( ) under Customers in the Navigator.

Task buttons for companies

Click the Actions button to access the following options:

  • Edit company: Click here to change the details registered for the company. See Edit companies.
  • New contact: Click here to create a new contact linked to the active company. See Create contacts.
  • Invoices: Click here to display invoice data recorded for the company. See View invoice data for companies.
  • Open in SuperOffice CRM: Click here to show the company in SuperOffice CRM, if you have a licence for this application.
