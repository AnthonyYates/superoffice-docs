View a script/macro

When you click a script or macro in the Macros and scripts tab, the Script screen opens. In addition to the information in the top part of the screen, you can find more script details in the Description and Script tabs.

To go to the Macros and scripts tab:

  • In SuperOffice Service: Select System settings > System design > CRMScript and go to the Macros and scripts tab.
  • In SuperOffice Settings and maintenance: Click CRMScript in the navigator and go to the Macros and scripts tab.

This screen has following options:

Icon Description
Macros and scripts: Click here to return to the list of the scripts and macros.
Edit script: Click here to edit the script or macro.
Run script: Click here to run the script or macro in standard mode.
Debug script: Click here to run the selected script or macro in debug mode. This opens the Debugger screen (see Debugger). If you run a script in debug mode, it stops at the first line. You can open a debug session from the Debug sessions tab. See Debug sessions.
Trace script: Click here to start tracing the script or macro. See Tracing.
