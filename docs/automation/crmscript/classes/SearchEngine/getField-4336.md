Returns the result of the field after been used on the function



* **field:** The name of the field
* **function:** The function to use on the field
   Possible values:
  FuncCount, FuncAvg, FuncSum, FuncMax, FuncMin, FuncHour,
  FuncWDay, FuncUpper, FuncLower
* **Returns:** The result of the funciton used on the field
