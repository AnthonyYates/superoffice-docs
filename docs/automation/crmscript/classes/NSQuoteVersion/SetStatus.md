If there was a problem with for instance calculation, this field is set to warning or error.

**Parameter:** 
 - **status** QuoteStatus
     - Enum: 0 = Ok 
     - Enum: 1 = OkWithInfo 
     - Enum: 2 = Warning 
     - Enum: 3 = Error
