Saves the tab order. The order is saved pr. user.

**Parameters:**
 - **tabOrder** Name of the tab control

**Returns:** This method has no return value

```crmscript
NSPreferenceAgent agent;
TabOrder tabOrder;
Void res = agent.SaveTabOrder(tabOrder);
```
