Saves a connection to the database.

**Parameters:**
 - **connection** The connection to save.

**Returns:** The resulting connection.

```crmscript
NSErpSyncAgent agent;
ErpConnection connection;
ErpConnection res = agent.SaveConnection(connection);
```
