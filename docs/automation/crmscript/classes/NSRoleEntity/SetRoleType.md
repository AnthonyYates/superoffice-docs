The role type. Note that changing this field has no effect, so treat this as a read-only field. 

**Parameter:** 
 - **roleType** Integer
     - Enum: 0 = Employee 
     - Enum: 1 = ExternalUser 
     - Enum: 2 = Anonymous 
     - Enum: 3 = System
