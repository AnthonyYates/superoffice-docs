The layout of the report.

**Parameter:** 
 - **reportLayout** Integer
     - Enum: 0 = Unknown 
     - Enum: 1 = List 
     - Enum: 2 = Label 
     - Enum: 3 = GroupList 
     - Enum: 4 = CrossTable 
     - Enum: 5 = CalendarWeek5 
     - Enum: 6 = CalendarWeek7 
     - Enum: 7 = CalendarMonth 
     - Enum: 8 = Text
