Preference level, using the standard enumerated type. Undefined is returned for preferences that do not have a current setting.

**Returns:** PreferenceLevel

     - Enum: 0 = Undefined 
     - Enum: 1 = HardDefault 
     - Enum: 2 = SystemWide 
     - Enum: 3 = Database 
     - Enum: 4 = Group 
     - Enum: 5 = Individual 
     - Enum: 6 = PC
