Preference level, using the standard enumerated type. Undefined is returned for preferences that do not have a current setting.
* Undefined=0
* HardDefault=1
* SystemWide=2
* Database=3
* Group=4	
* Individual=5
* PC=6

* **p_0:** Integer
* **Returns:** Void
