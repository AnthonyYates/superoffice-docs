The confidentiality of appointments is shown as different types of “private” on the appointment. For an updated list of “private” types, see the database manual.

**Parameter:** 
 - **private** Integer
     - Enum: 0 = Public 
     - Enum: 1 = PrivateUser 
     - Enum: 2 = PrivateGroup
