Appointment Completed state. This property is the part of the Status property that is the completed state. Could be three state if the three state user preference is set.

**Parameter:** 
 - **completed** Integer
     - Enum: 0 = Unknown 
     - Enum: 1 = NotStarted 
     - Enum: 2 = Started 
     - Enum: 3 = Completed
