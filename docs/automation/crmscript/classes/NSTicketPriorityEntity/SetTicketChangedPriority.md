This field indicates what to do with the escalation chain when the request is changed into this priority

**Parameter:** 
 - **ticketChangedPriority** TicketPriorityEscalateEvent
     - Enum: -1 = None 
     - Enum: 0 = Stop 
     - Enum: 1 = Continue 
     - Enum: 2 = Restart
