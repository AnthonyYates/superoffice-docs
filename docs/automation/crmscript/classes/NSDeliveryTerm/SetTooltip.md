Set the tooltip description for the DeliveryTerm list item.

**Parameter:** 
 - **tooltip** String DeliveryTerm tooltip description.

```crmscript
NSDeliveryTerm thing;
String descr = "something new";
thing.SetTooltip(descr);
```
