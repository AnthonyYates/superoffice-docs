Loads the extra table entry with the given id



###Parameter:###


 - id: Id of the extra table entry you want to load.


Returns true if the entry was found and loaded.



###Example code:###


    ExtraTable ex;
    
    Bool b = ex.load(2);
    print(b.toString());
