Webhook status - should we post events to the URL? 1=Active, 2=Stopped or 3=TooManyErrors

**Parameter:** 
 - **state** WebhookState
     - Enum: 0 = Unknown 
     - Enum: 1 = Active 
     - Enum: 2 = Stopped 
     - Enum: 3 = TooManyErrors
