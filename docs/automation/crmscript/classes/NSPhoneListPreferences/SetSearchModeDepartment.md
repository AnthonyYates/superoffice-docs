The Department search type, e.g. BeginsWith, Contains, Endswith, Matches

**Parameter:** 
 - **searchModeDepartment** Integer
     - Enum: 0 = Exact 
     - Enum: 1 = BeginsWith 
     - Enum: 2 = EndsWith 
     - Enum: 3 = Contains
