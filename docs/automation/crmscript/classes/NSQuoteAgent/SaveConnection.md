Saves a connection to the database.

**Parameters:**
 - **connection** The connection to save.

**Returns:** The resulting connection.

```crmscript
NSQuoteAgent agent;
QuoteConnection connection;
QuoteConnection res = agent.SaveConnection(connection);
```
