    String get(String field, Integer fieldMask)
    

Get the value of the field accoring to the wanted fieldMask


* **field:** The name of the field to select
* **fieldMask:** Possible values:
1: Id of the table
2: Data value
4: Display value
8: Search value
16: Sort value
* **Returns:** The value of the field
