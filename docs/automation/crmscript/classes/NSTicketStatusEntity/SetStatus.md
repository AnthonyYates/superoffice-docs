The &apos;classic&apos; ticket status. I.e. active/closed/postponed/deleted

**Parameter:** 
 - **status** TicketBaseStatus
     - Enum: 0 = Unknown 
     - Enum: 1 = Active 
     - Enum: 2 = Closed 
     - Enum: 3 = Postponed 
     - Enum: 4 = Deleted 
     - Enum: 5 = Merged
