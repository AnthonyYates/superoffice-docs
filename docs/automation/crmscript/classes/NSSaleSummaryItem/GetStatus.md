The sale's status, indicating whether the sale is open(1), sold(2) or lost(3).

**Returns:** Integer

     - Enum: 0 = Unknown 
     - Enum: 1 = Open 
     - Enum: 2 = Sold 
     - Enum: 3 = Lost 
     - Enum: 4 = Stalled 
     - Enum: 1000 = SaintAll
