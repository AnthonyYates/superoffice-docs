Icon hint, which should resolve to an icon that is compatible in size with a standard menu or dropdown list font. May be blank.

**Parameter:** 
 - **iconHint** String

```crmscript
NSCommandInfo thing;
String iconHint;
thing.SetIconHint(iconHint);
```
