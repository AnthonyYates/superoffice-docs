# Variables for quote lines with subscription

| Variable | Description |
|----------|-------------|
| qsqa | Subscription quantity (Blank if not subscription) |
| qsub | Is subscription? ("Yes" or blank) |
| qsun | Subscription unit (Blank if not subscription) |
