## Adding columns, text blocks, images etc

|   |   |
|---|---|
|![icon](media/editor-sidepanel-btn-content.png) | From the **Content** section you can add all types of content, such as columns, text blocks, images, menus/buttons and SoMe links. Drag and drop content from this section to the message. Click the content to edit it in the side panel. See [Edit content](Edit-a-paragraph.md).|

| Icon | Name | Function |
|------|------|----------|
| ![icon](media/editor-sidepanel-btn-content-columns.png) | Columns | Use columns to structure your message into columns and rows.   |
| ![icon](media/editor-sidepanel-btn-content-button.png) | Button  | Use buttons to add tracked links, regular URLs, or subscribe links. |
| ![icon](media/editor-sidepanel-btn-content-divider.png) | Divider | Use divider lines to separate text blocks and other content. |
| ![icon](media/editor-sidepanel-btn-content-Heading.png) | Heading | Add a heading to your message. |
| ![icon](media/editor-sidepanel-btn-content-HTML.png) | HTML    | Add html formatted text to your message. |
| ![icon](media/editor-sidepanel-btn-content-image.png) | Image   | Add images from the SuperOffice database or upload you own images. Insert images in the message. |
| ![icon](media/editor-sidepanel-btn-content-menu.png) | Menu    | Add a menu with multiple links. |
| ![icon](media/editor-sidepanel-btn-content-social.png) | Social  | Add links to social media. |
| ![icon](media/editor-sidepanel-btn-content-text.png) | Text    | Add text content. See Edit content. |
| ![icon](media/editor-sidepanel-btn-content-Receipt.png) | Receipt | Add a receipt showing the information that the recipient has added to a form. |

## Adding blocks ![icon](media/editor-sidepanel-btn-blocks.png)

This section contains saved content blocks such as headers and footers. Drag and drop content from this section to the message.

To save a content block, click a row and select **Save block** (![icon](media/NEWbtn-SaveBlock.png)).

## Editing global layout ![icon](media/editor-sidepanel-btn-body.png)

In this section you can edit the global properties and layout for the message.

> [!NOTE]
> If you have defined specific properties for individual sections, these are not overridden by global properties.

## Adding images

Add images from the SuperOffice database or from the royalty-free online image database. See [Insert images in the message](Insert-images-in-the-message.md).
