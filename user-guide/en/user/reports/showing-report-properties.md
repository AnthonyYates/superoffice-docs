# Show report properties

To view the properties of individual reports in the **Reports** screen:

1. Open the **Reports** screen.
    <!-- Fix reuse ID=a1 -->
    [!include[How to-goto-reports](../includes/goto-reports.md
)]
2. Click one of the tabs to display the reports you want.

> [!TIP]
> You can go directly to a specific tab in the **Reports** screen by clicking the word **Reports** next to the **Reports** button in the Navigator and then selecting from the list.

Favourite reports

> [!NOTE]
> For more details, see [Favourite reports][5].

Standard reports

| Type | Description |
|---|---|
| Company/contact | shows company/contact reports. |
| Project | shows project reports. |
| Sale | shows sales reports. |
| Activity | shows reports for follow-ups and documents. |
| Selection | shows selection reports. |

> [!NOTE]
> For more details, see [Standard reports][1].

1. Select the required report from the report list in the top left of the window. Note that:

2. The field on the right of the report list displays a description of the selected report.
    * The field at the bottom displays a list of the criteria that the report uses. These are described under [Search criteria for reports][2]. (See also [Criteria list][3].)
    * The thumbnail in the top right corner changes according to the type of report you selected:

## Report types

| Icon | Description |
|---|---|
| ![icon][img1] | **Label**: this report type is designed for displaying and printing labels. |
| ![icon][img2] | **List**: this report type is designed for previewing and printing ordinary lists. |
| ![icon][img3] | **Group list**: this report type is designed for previewing and printing group lists. |
| ![icon][img4] | **Calendar**: this report type is designed for displaying and printing various calendar views. |
| ![icon][img5] | **Cross table**: this report type is designed for tables, spreadsheets and charts. |
| ![icon][img6] | **Text export**: this report type is designed for exporting to text files. |

> [!TIP]
> To see what a finished report will look like, follow the procedure under [View reports][4].
