# Create a selection

In SuperOffice Service you can create different types of selections. Click the appropriate link below, depending on the type of selection you want to create:

* [Create a dynamic selection][1]

    > [!TIP]
    > A simple way of creating a dynamic selection is to base it on search criteria. See [Create a selection based on search criteria][2].

* [Create a script-based selection][3]

* [Create a static selection][4]

* [Create a combined selection][5]

To learn more about the various selection types, see [Selections][6].
