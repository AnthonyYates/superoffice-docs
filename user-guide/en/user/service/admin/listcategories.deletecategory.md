# Delete categories

To delete a category:

1. Select ![icon][img1] **System settings** &gt; **Categories**.

2. Click the category name. This takes you to the **Category properties** screen.

    > [!TIP]
    > If the category is a subcategory, you first need to click the plus sign to the left of the parent category in order to display the associated subcategories.

3. Click the **Delete** button at the bottom of the screen. The **Delete category** screen appears.

4. In the **Move messages to** list, specify where to move the requests in this category.

5. Click **Delete**.

6. Click **OK** when you are asked if you want to delete the category. The category is deleted and all the requests are moved to the specified alternative category.
