# Delete selections

To delete an existing selection:

1. Open the **Selection** screen.
    <!-- Fix reuse ID=a1 -->
    [!include[Howto](../includes/goto-selections.md)]
2. Click the **Task** button.
3. Select **Delete this selection**.
4. When you are asked if you really want to delete the selection, click **Yes**. (For more details about this, see [Delete items in SuperOffice CRM][1].)
