# Section tabs in the Company screen

The **Company** screen contains the following section tabs:

* [Contacts][1]: The **Contacts** section tab contains information about employees of the active company.
* [Relations][2]: The **Relations** section tab displays any connections existing between companies and/or contacts.
* [Activities][3]: The **Activities** section tab contains all the activities linked to the active company.
* [Projects][4]: The **Projects** section tab shows all the projects for the active company.
* [Sales][5]: The **Sales** section tab contains information about any sales connected to the company.
* [Requests][6]: The **Requests** section tab shows SuperOffice Service requests for the active company.
* [ERP][7]: Here you can link a SuperOffice company to a customer/supplier in the ERP system.
