# Global buttons

The global buttons appear throughout the SuperOffice Service interface:

| Button | Description |
|:-:|---|
| ![icon](../media/btn_add.png)                | Click here to add information, such as e-mail addresses or new contacts or companies. |
| ![icon](../media/btn_Menu.png)               | Click here to open a menu containing tasks relevant to the dialog or screen. |
| ![icon](../media/btn_dropdown.png)           | Click here to open and close a list box. |
| ![icon](../media/favourite_inactive.png)     | Click here to add the current record (request, contact or similar) as a favourite. See [Favourites](rms.navigate.favourites.md). |
| ![icon](../media/search.png)                 | Click here to start a search. See [Search in SuperOffice Service](rms.search.md). |
| ![icon](../media/btn_editelementprofile.png) | Click here to configure the appearance and behaviour of a screen element (such as a list of search results, a properties view or a request header). See [Profile](admin.listProfiles.md). |
| ![icon](../media/btn_calendar.jpg)           | Click here to select a date from the calendar. |
