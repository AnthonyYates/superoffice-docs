# Statistics

You have the following options in the statistics module:

* [Standard reports](stat.newStat.standard.md) – provides a set of general reports
* [Request statistics](stat.simpleStat.md) – provides various statistics on requests in the system
* [Login statistics](stat.loginStat.md) – provides an overview of the time spent by requests handlers in the system
* [Invoice statistics](stat.invoiceStat.md) – provides an overview of all items to be invoiced
