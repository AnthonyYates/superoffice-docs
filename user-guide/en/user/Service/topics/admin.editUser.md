# Personal settings

In the **Edit users** screen, you can adjust the settings defined for your user profile and which control your use of SuperOffice Service.

To edit your user settings, select ![icon](../media/globalmenu-PersonalSettings-small.png)**&gt; Settings**. The following tabs are available:

[User information](admin.editUser.helpUserInfo.md)

[Signature](admin.editUser.helpSignature.md)

[Interface](admin.editUser.helpInterface.md)

[Preferences](admin.editUser.helpSettings.md)

[Category membership](admin.editUser.helpCategoryMembership.md)

[Other groups](admin.editUser.helpOthergroups.md)

[Password](admin.editUser.helpPassword.md) ![icon](../../media/OnPremises.png)

[Notification](admin.editUser.helpNotification.md)

[Extra fields](admin.editUser.helpExtrafields.md)
