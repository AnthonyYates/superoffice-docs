# Delete escalation levels

To delete an escalation level from SuperOffice Service, do as follows:

1. Select ![icon](../media/globalmenu_settings_small.png)**System settings** &gt; **Priorities**. The **Priorities** screen appears.
2. Click on the escalation level you are interested in. This takes you to the **Escalation level properties** screen.
3. Click the **Delete** button at the bottom of the screen. The escalation level is removed from SuperOffice Service.
