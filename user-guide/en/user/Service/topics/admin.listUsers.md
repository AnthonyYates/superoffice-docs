# Users

**Note:** If this option is not visible, that means you have insufficient rights for this function. Contact your system administrator for more information.

Selecting ![icon](../media/globalmenu_settings_small.png)**System settings &gt; Users** takes you the **Users** screen, where you have the following options:

* In the **Users** tab, you can see all the individual users. See [Users](admin.listUsers.listUsers.md).
* In the **User groups** tab, you can see the existing user groups. See [User groups](admin.listUsers.listUsergroups.md).

To create, delete or restore users, go to SuperOffice Settings and maintenance.
