# Edit role

To edit the information already recorded for a role:

1. Select ![icon](../../../media/icons/globalmenu-settings-small.png)**System settings &gt; Roles**.
2. Click the required role name. The **Edit role** screen appears.
3. Make the required changes. For more information about the tabs and fields in this screen, see [Create roles](admin.listRoles.newRole.md).
4. Click **OK**. The changes are saved.
