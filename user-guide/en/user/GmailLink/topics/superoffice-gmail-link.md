# SuperOffice Gmail Link

SuperOffice Gmail Link is a plug-in which links the Google Gmail client with SuperOffice CRM and/or SuperOffice Service. This means that among other things, you can:

* Archive e-mails and attachments as documents in SuperOffice CRM.
* Create activities in SuperOffice CRM based on e-mails in Gmail.
* Create requests in SuperOffice Service based on e-mails in Gmail.
* Look up senders in SuperOffice.

To install SuperOffice Gmail Link, go to the [SuperOffice App Store][1].
