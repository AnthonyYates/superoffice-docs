# Service_KB_FolderAdmin_delete

<p>When you are ready to delete the folder itself, do as follows:</p>
<ol style="list-style: decimal;">
<li>Go to the required folder.</li>
<li>Click the <span class="UI-text">Edit folder</span> button.</li>
<li>Click <span class="UI-text">Delete</span>. The folder is deleted 
 immediately.</li>
</ol>
