# Document_dialog_mainpart

<p style="font-weight: bold;"><span style="font-weight: bold;">[Document 
 template]</span></p>
<p>Click the arrow ( <img src="StandardCRM/bitmaps/Dialogboksknapper_pil ned.bmp" 
 alt="" style="border: none;" border="0" /> 
 ) next to the type field at the top of the dialog to display a list of 
 document templates that you can choose from. The available templates are 
 defined in SuperOffice Settings and maintenance. Here you can also use 
 <a href="StandardCRM/chap01/Using_FastSearcher.htm">FastSearcher</a>.</p>
<?rh-cbt_start condition="Help" ?><p class="Note"><span style="font-weight: bold;">Note:</span> 
 The templates may have been created in various programs, for example, 
 Word or Excel, so when you create a document (see <a href="StandardCRM/chap02/Creating_new_documents.htm">Create 
 new documents</a>), the template is opened in the program it was created 
 in. This means, for example, that DOCX files will open in Word.</p><?rh-cbt_end ?>
<p>&#160;</p>
<p style="font-weight: bold;">[Company]</p>
<p>Enter here the name of the company you are writing to. You can search 
 directly from this field.</p>
<p>&#160;</p>
<p style="font-weight: bold;">[Contact]</p>
<p>Click the arrow next to this field and a list of known contacts in the 
 company is displayed. <?rh-cbt_start condition="Print" ?>Select a contact.<?rh-cbt_end ?></p>
<p>&#160;</p>
<p style="font-weight: bold;">[Project]</p>
<p>Enter the name of the project that the document is linked to. <?rh-cbt_start condition="Help" ?>You 
 can search directly from this field.<?rh-cbt_end ?></p>
<p>&#160;</p>
<p style="font-weight: bold;">[Sale]</p>
<p>Here you enter the name of the sale, if any, that the document is linked 
 to. <?rh-cbt_start condition="Help" ?>You can search directly from this 
 field.<?rh-cbt_end ?></p>
<p>&#160;</p>
<p style="font-weight: bold;">[Type subject here]</p>
<p>Enter here the text to be used as a heading for the document. This is 
 the text which is displayed in section tabs and reports where the document 
 is listed.</p>
<p>&#160;</p>
<p style="font-weight: bold;">Language</p>
<p>Select a language for the template. This option is only available if 
 the selected template includes translated versions.</p>
<p>&#160;</p>
<p style="font-weight: bold;"><span style="font-weight: bold;">Date</span></p>
<p>Today's date is displayed automatically in this field. If you click 
 the arrow next to the <span style="font-weight: bold;">Date</span> field, 
 this brings up a monthly calendar where you can select the date you require.</p>
<p class="Note"><span style="font-weight: bold;">Note:</span> You can only 
 change the date when you create the document. You cannot access this field 
 in edit mode.</p>
<p>&#160;</p>
<p style="font-weight: bold;"><span style="font-weight: bold;">Our ref.</span></p>
<p>Enter your own reference. The default entry for each template is defined 
 in SuperOffice Settings and maintenance, but you can also edit the content 
 of this field.</p>
<p>&#160;</p>
<p style="font-weight: bold;">Your ref.</p>
<p>Enter the addresseeâ€™s reference from their communication.</p>
<p>&#160;</p>
<p style="font-weight: bold;">Visible for</p>
<p>Click the <span style="font-weight: bold;">Visible for</span> field 
 to display a list that allows you to choose whether the document will 
 be visible to all users, private (i.e. visible only to the user defined 
 as owner), or visible to all users in one of the user groups that the 
 owner belongs to.</p>
<p class="Note"><span style="font-weight: bold;">Note:</span> If you specify 
 that a document will only be visible to the owner, it will appear in italics 
 on the <span style="font-weight: bold;">Activities</span> section tab.</p>
<p>&#160;</p>
<p style="font-weight: bold;">Checked out for editing</p>
<p>If the document has been checked out for editing, one of the following 
 icons are displayed:</p>
<p><img src="StandardCRM/bitmaps/documentlock_editing.png" alt="" style="border: none;" 
 border="0" /> The document is currently checked out for editing 
 by you.</p>
<p><img src="StandardCRM/bitmaps/documentlock_locked.png" alt="" style="border: none;" 
 border="0" /> The document is currently checked out for editing 
 by another user.</p>
<p class="Note"><span style="font-weight: bold;">Tip</span>: Hold the mouse 
 pointer over an icon to show information about the user who has checked 
 out the document.</p>
<p>&#160;</p>
<p style="font-weight: bold;"><span style="font-weight: bold;">Completed</span></p>
<p>By default, the document is marked as completed ( <img src="StandardCRM/bitmaps/followup_completed_SMALL.png" 
 alt="" style="border: none;" 
 border="0" /> 
 ), but you can remove this check mark if you wish ( <img src="StandardCRM/bitmaps/followup_notcompleted_SMALL.png" 
 alt="" style="border: none;" 
 border="0" /> 
 ). The document will then be displayed on the <span style="font-weight: bold;">Activities</span> 
 section tab in the <span style="font-weight: bold;">Diary</span> screen 
 until it is marked as completed.</p>
<?rh-cbt_start condition="Help" ?><p class="Note"><?rh-cbt_start condition="Help" ?><span 
 style="font-weight: bold;">Note:</span><?rh-cbt_end ?> You cannot 
 edit a completed document until you have undone its <?rh-cbt_start condition="Help" ?><span 
 style="font-weight: bold;">Completed</span><?rh-cbt_end ?> status. 
 See <a href="StandardCRM/chap02/Marking_activities_as_completed_not_completed.htm">Change 
 Completed status of an activity</a>.</p><?rh-cbt_end ?>

<?rh-cbt_start condition="Help" ?><p style="font-weight: bold;">[User-defined 
 fields]</p><?rh-cbt_end ?>
<?rh-cbt_start condition="Help" ?><p>Here, three of the fields from the 
 <span style="font-weight: bold;">More</span> tab may be displayed (see 
 <?rh-cbt_start condition="Help" ?><a href="StandardCRM/chap02/More_tab_Document.htm">The 
 More tab</a><?rh-cbt_end ?>), as defined in SuperOffice Settings and maintenance. 
 The fields displayed can vary from user group to user group.</p><?rh-cbt_end ?>
<?rh-cbt_start condition="Help" ?><p class="Note"><span style="font-weight: bold;">Note:</span> 
 User group membership is defined in the <span style="font-weight: bold;">Users</span> 
 screen in SuperOffice Settings and maintenance.</p><?rh-cbt_end ?>
<?rh-cbt_start condition="Help" ?><p>&#160;</p><?rh-cbt_end ?>
<?rh-cbt_start condition="Help" ?><p style="font-weight: bold;">Task</p><?rh-cbt_end ?>
<?rh-cbt_start condition="Help" ?><p>When you click this button, the following 
 options are displayed:</p><?rh-cbt_end ?>
<?rh-cbt_start condition="Help" ?><ul style="list-style: disc;">
<li class="p"><?rh-cbt_start condition="Help" ?><span style="font-weight: bold;">Delete</span><?rh-cbt_end ?>: 
 Deletes the current entry.</li>
<li class="p"><?rh-cbt_start condition="Help" ?><span style="font-weight: bold;">Help</span><?rh-cbt_end ?>: 
 Opens the help for the active dialog.</li>
<li class="p"><span style="font-weight: bold;">Copy shortcut</span>: 
 Copies a shortcut to the active entry. See <?rh-cbt_start condition="Help" ?><a 
 href="StandardCRM/chap03/Creating_shortcuts_to_entries_in_CRM.htm">Create 
 shortcuts to entries in SuperOffice CRM</a><?rh-cbt_end ?>.</li>
<li class="p"><span style="font-weight: bold;">Revert to saved version</span>: 
 Select this option to cancel a check-out. This may be relevant if 
 you have checked out the document by mistake and are unable to check 
 it in again. See <a href="StandardCRM/chap02/doc_checkoutin.htm">Check 
 in/out documents</a>.</li>
</ul><?rh-cbt_end ?>
<?rh-cbt_start condition="Help" ?><p>Below these options are shortcuts 
 to other relevant tasks that can be performed on the active entry. In 
 SuperOffice Settings and maintenance, you can also define your own tasks 
 and add them under the <span style="font-weight: bold;">Task</span> button. 
 For more information about this, see the Help for SuperOffice Settings 
 and maintenance.</p><?rh-cbt_end ?>
