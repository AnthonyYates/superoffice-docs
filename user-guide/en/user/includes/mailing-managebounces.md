# MAILING_managebounces

<p><span class="UI-text">Export</span></p>
<ol style="list-style: decimal;">
<li>Select the e-mail addresses you want to export.</li>
<li>Select <span class="UI-text">Actions</span> &gt; <span class="UI-text">Export</span>.</li>
<li>Select an export format. The export file displays in the bottom 
 of you browser. It can also be found in you downloads folder.</li>
</ol>
<p>&#160;</p>
<p><span class="UI-text">Add to new selection:</span></p>
<ol style="list-style: decimal;">
<li>Select the e-mail addresses you want to add to a new selection.</li>
<li>Select <span class="UI-text">Actions</span> &gt; <span class="UI-text">Add 
 to new selection</span>.</li>
<li>Enter the name of the new selection and click <span class="UI-text">OK</span>. 
 A new static selection is created with the selected e-mail addresses.</li>
</ol>
<p>&#160;</p>
<p><span class="UI-text">Add to new project</span></p>
<ol style="list-style: decimal;">
<li>Select the e-mail addresses you want to add to a new project.</li>
<li>Select <span class="UI-text">Actions</span> &gt; <span class="UI-text">Add 
 to new project</span>.</li>
<li>Enter the name of the new project and click <span class="UI-text">OK</span>. 
 A new project is created with the selected e-mail addresses.</li>
</ol>
<p>&#160;</p>
<p><span class="UI-text">Add to existing selection</span></p>
<ol style="list-style: decimal;">
<li>Select the e-mail addresses you want to add to an existing selection.</li>
<li>Select <span class="UI-text">Actions</span> &gt; <span class="UI-text">Add 
 to existing selection</span>.</li>
<li>Start typing in the search field to find the relevant selection.</li>
<li>In the list of results, click the selection you want to use. </li>
<li>Click <span class="UI-text">OK</span>. The selected e-mail addresses 
 are added to the selection.</li>
</ol>
<p>&#160;</p>
<p><span class="UI-text">Add to existing project</span></p>
<ol style="list-style: decimal;">
<li>Select the e-mail addresses you want to add to an existing project.</li>
<li>Select <span class="UI-text">Actions</span> &gt; <span class="UI-text">Add 
 to existing project</span>.</li>
<li>Start typing in the search field to find the relevant project.</li>
<li>In the list of results, click the project you want to use. </li>
<li>Click <span class="UI-text">OK</span>. The selected e-mail addresses 
 are added to the project</li>
</ol>
<p>&#160;</p>
<p><span class="UI-text">Delete selected e-mail addresses</span></p>
<p>Use this option to permanently delete e-mail addresses from the database.</p>
<ol style="list-style: decimal;">
<li>Select the e-mail addresses you want to delete.</li>
<li>Select <span class="UI-text">Actions</span> &gt; <span class="UI-text">Delete 
 selected e-mail addresses</span>.</li>
<li>Click <span class="UI-text">OK</span> to confirm.</li>
</ol>
<p>&#160;</p>
<p><span class="UI-text">Clear bounce-flag</span></p>
<p style="font-weight: normal;"><span class="UI-text" style="font-weight: normal;">Use 
 this option to remove e-mail addresses from the list, so they can be used 
 in mailings. This may be relevant if bounces are due to </span>wrong configuration 
 or temporary technical reasons, or if you know that these e-mail addresses 
 are correct and want to try sending a second time.</p>
<ol style="list-style: decimal;">
<li>Select the e-mail addresses that you want to clear of bounce-flags.</li>
<li>Select <span class="UI-text">Actions</span> &gt; <span class="UI-text">Clear 
 bounce-flag</span>.</li>
<li>Click <span class="UI-text">OK</span> to confirm.</li>
</ol>
<p>&#160;</p>
<p><span class="UI-text">Set as former employee</span></p>
<p style="font-weight: normal;"><span class="UI-text" style="font-weight: normal;">Use 
 this option to set a contact as a former employee.</span></p>
<ol style="list-style: decimal;">
<li><p>Select the e-mail addresses you want to delete.</p></li>
<li>Select <span class="UI-text">Actions</span> &gt; <span class="UI-text">Set 
 as former employee</span>.</li>
<li>Click <span class="UI-text">OK</span> to confirm.</li>
</ol>
