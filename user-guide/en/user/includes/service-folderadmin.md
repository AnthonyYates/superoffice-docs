# SERVICE_FolderAdmin

<p class="Prosedyre">Create folders</p>
<p>To create a new folder:</p>
<ol style="list-style: decimal;">
<li>Click <span class="UI-text">New folder</span> at the top of the 
 screen.</li>
<li>In the <span class="UI-text">Folder</span> field, select a parent 
 folder in the list.</li>
<li>Enter a descriptive name for the folder in the <span class="UI-text">Name</span> 
 field.</li>
<li>Click <span class="UI-text">OK</span>. The new folder is created. 
 </li>
</ol>
<p class="Prosedyre">Edit folders</p>
<p>You can edit both the name of a folder and the details you entered when 
 creating it. To do this:</p>
<ol style="list-style: decimal;">
<li>Go to the required folder.</li>
<li>Click the name of the folder.</li>
<li>Amend the fields as described above.</li>
<li>Click <span class="UI-text">OK</span>. The changes are saved.</li>
</ol>
<p class="Prosedyre">Delete folders</p>
<p>When you delete a folder, all its contents (selections, FAQ entries, 
 reply templates, tables, scripts, macros etc.) are also deleted from <?rh-udv_start 
 name="ProdName_CS" ?>SuperOffice Service<?rh-udv_end ?>. To do this: 
 </p>
<ol style="list-style: decimal;">
<li>Go to the required folder.</li>
<li>Click the name of the folder.</li>
<li>Click <span class="UI-text">Delete</span>.</li>
<li>Click <span class="UI-text">OK</span> when you are asked if you 
 want to delete the folder. The folder and its contents are removed 
 from SuperOffice Service.</li>
</ol>
