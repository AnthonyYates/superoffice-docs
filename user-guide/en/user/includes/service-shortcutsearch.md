# Service_ShortcutSearch

<p class="Prosedyre">Shortcut search</p>
<p>To quickly access a specific record (such as a request or contact), 
 function or screen in SuperOffice Service, 
 use the &quot;shortcut search&quot;:</p>
<ol style="list-style: decimal;">
<li>Press <span style="font-weight: bold;">CTRL+SPACE</span> on your 
 keyboard. The shortcut popup shows a fast searcher and all available 
 menu items from the navigator, personal settings and system settings.</li>
<li>Enter a search term, such as a function name, request ID, contact 
 name or menu item. The result updates dynamically as you type.</li>
<li>Select the result you want to open.</li>
</ol>
<p class="Note"><span style="font-weight: bold;">Tip</span>: You can also 
 use the <a href="Service/topics/rms.search.useQuickSearch.html#SearchForOtherRecords">keywords 
 used in the quick search</a> to search for specific items like contacts 
 or companies.</p>
