# MAILING_Step2_Layout_form_email

<p>In the second step, <span class="UI-text">Template</span>, you have 
 to decide what your mailing should look like. You do this by selecting 
 an existing template, a previous mailing, or a template from the online 
 library.</p>
<p class="Note"><span style="font-weight: bold;">Tip:</span> Save yourself 
 some time and create your mailing using one of the free templates in the 
 <span class="UI-text">Online template library</span>. The Online template 
 library in <?rh-udv_start name="ProdName_eM" ?>SuperOffice Marketing<?rh-udv_end ?> 
 contains a wide selection of professionally designed message template. 
 Free for use!</p>
<p>In order to do this:</p>
<ol style="list-style: decimal;">
<li>Select a template in one of the following ways: 
<ul style="list-style: disc;">
<li>Select one of the available tabs (<span class="UI-text">Online 
 template library</span>, <span class="UI-text">Recent templates</span>, 
 <span class="UI-text">All templates</span>, etc) to view available 
 mailings or templates on which to base your new mailing.</li>
<li>Enter a search phrase in <span class="UI-text">Search templates</span> 
 and press <span class="UI-text">ENTER</span> or click the search 
 icon ( <img src="Mailing/Bitmaps/btn_Search.png" alt="" style="border: none;" 
 border="0" /> ). </li>
</ul></li>
<li>In the list below, click a template to select it. 
<p class="Note"><span style="font-weight: bold;">Note:</span> If you 
 are using a template from the Online template library, you need to 
 click <span class="UI-text">Use this template</span> in the template 
 preview window. You are automatically redirected to the <span class="UI-text">Content</span> 
 step.</p></li>
<li>Click <span class="UI-text">Next</span> to go to the next step.</li>
</ol>
