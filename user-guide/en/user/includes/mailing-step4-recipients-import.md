<p class="Prosedyre">Import a list of recipients (contacts) from an external 
 file</p>
<p>You can import recipients from an <a href="Mailing/Helptopics/Create_address_list.htm">external 
 comma-separated file</a> (.txt, .csv or similar). </p>
<p class="Note"><span style="font-weight: bold;">Note:</span> When you 
 import a list of names and e-mail addresses, <?rh-udv_start name="ProdName_eM" ?>SuperOffice Marketing<?rh-udv_end ?> 
 will try to locate the imported recipients in the SuperOffice database. 
 If matches are found, the recipients' companies will be listed in the 
 <span class="UI-text">Company</span> column in the <span class="UI-text">Recipients</span> 
 list. Any names/e-mail addresses that are <span style="font-weight: bold;">not</span> 
 found in SuperOfffice, will be filtered out and not included in the mailing. 
 This is due to GDPR regulations.</p>
<p>To do this:</p>
<ol style="list-style: decimal;">
<li>Click <span class="UI-text">Select file</span>.</li>
<li>Locate the relevant file (.txt, .csv or similar text format).</li>
<li>Click <img src="Mailing/Bitmaps/btn_Add.png" alt="" style="border: none;" 
 border="0" />. The list is added to the <span class="UI-text">Recipient 
 sources</span> list, and the <span class="UI-text">Recipients</span> 
 list is updated.</li>
<li>Repeat steps 2-4 to add more recipients. Additional recipients 
 will be added to the same row in the <span class="UI-text">Recipient 
 sources</span> list. The total number of recipients will display in 
 the <span class="UI-text">Name</span> column.</li>
</ol>
