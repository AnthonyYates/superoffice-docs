<p><span style="font-weight: bold;">Recurrence</span></p>
<p>In this field, you indicate whether the follow-up is a repeating one. 
 Click the word <span style="font-weight: bold;">Never</span> to the right 
 of the <span style="font-weight: bold;">Recurrence</span> field if you 
 want the appointment to be repeated at regular intervals. <?rh-cbt_start condition="Help" ?>See 
 <?rh-cbt_end ?><?rh-cbt_start condition="Help" ?><a href="StandardCRM/chap02/Creating_recurring_follow-ups.htm">Create 
 repeating follow-ups</a><?rh-cbt_end ?><?rh-cbt_start condition="Help" ?> 
 for more information. If a follow-up has been specified as repeating, 
 an icon ( <?rh-cbt_end ?><?rh-cbt_start condition="Help" ?><img src="StandardCRM/bitmaps/Recurring_booking_assignment.bmp" 
 alt="" style="border: none;" border="0" /><?rh-cbt_end ?><?rh-cbt_start condition="Help" ?> 
 ) at the upper left of the main part of the dialog.<?rh-cbt_end ?></p>
<p>&#160;</p>
<p><span style="font-weight: bold;">Priority</span></p>
<p>Specify here the priority of the follow-up. Click the arrow to display 
 a list with the default options of high, medium and low priority. These 
 are defined in SuperOffice Settings and maintenance.</p>
<p>&#160;</p>
<p><span style="font-weight: bold;">Alarm</span></p>
<p>For notification of the follow-up, check <span style="font-weight: bold;">Alarm</span>.&#160;<?rh-cbt_start condition="Help" ?>For 
 more information about this, see <?rh-cbt_end ?><?rh-cbt_start condition="Help" ?><a 
 href="StandardCRM/chap02/Setting_an_alarm_for_follow-ups.htm">Set 
 alarms for follow-ups</a><?rh-cbt_end ?><?rh-cbt_start condition="Help" ?>. 
 If an alarm has been specified for a follow-up, an icon ( <?rh-cbt_end ?><?rh-cbt_start condition="Help" ?><img 
 src="StandardCRM/bitmaps/followup_details_alarm.bmp" alt="" style="border: none;" 
 border="0" /><?rh-cbt_end ?><?rh-cbt_start condition="Help" ?> ) is 
 displayed at the upper left of the main part of the dialog.<?rh-cbt_end ?></p>
<p>&#160;</p>
<p style="font-weight: bold;">Owner</p>
<p>Here you specify who owns the follow-up. The default is the user currently 
 logged in. If you click the name next to the <span style="font-weight: bold;">Owner</span> 
 field, you can transfer the appointment to another SuperOffice user. <?rh-cbt_start condition="Help" ?>The 
 <?rh-cbt_end ?><?rh-cbt_start condition="Help" ?><span style="font-weight: bold;">Assign 
 to</span><?rh-cbt_end ?><?rh-cbt_start condition="Help" ?> dialog opens. 
 Under the <?rh-cbt_end ?><?rh-cbt_start condition="Help" ?><span style="font-weight: bold;">Assign 
 this follow-up to</span><?rh-cbt_end ?><?rh-cbt_start condition="Help" ?> 
 field is a list box where you can select the required user group. Select 
 the required user and click <?rh-cbt_end ?><?rh-cbt_start condition="Help" ?><span 
 style="font-weight: bold;">OK</span><?rh-cbt_end ?><?rh-cbt_start condition="Help" ?> 
 to save the changes and close the dialog.<?rh-cbt_end ?></p>
<?rh-cbt_start condition="Help" ?><p class="Note"><?rh-cbt_start condition="Help" ?><span 
 style="font-weight: bold;">Note:</span><?rh-cbt_end ?> If a user is 
 already occupied at the time in question, the word <?rh-cbt_start condition="Help" ?><span 
 style="font-weight: bold;">CONFLICT</span><?rh-cbt_end ?> is displayed 
 in the <?rh-cbt_start condition="Help" ?><span style="font-weight: bold;">Status</span><?rh-cbt_end ?> 
 field. If you want to hide users with conflicts, check <?rh-cbt_start condition="Help" ?><span 
 style="font-weight: bold;">Hide conflict</span><?rh-cbt_end ?> at 
 the bottom of the dialog.</p><?rh-cbt_end ?>
<p>&#160;</p>
<p style="font-weight: bold;">Show as</p>
<p>Check <span style="font-weight: bold;">Free</span> or <span style="font-weight: bold;">Busy</span> 
 in the <span style="font-weight: bold;">Show as</span> field to specify 
 whether you are available to others while the appointment is taking place. 
 If you select <span style="font-weight: bold;">Free</span>, you can have 
 other follow-ups simultaneously, for example if you have reserved the 
 majority of the day to work on a project but you still have time for short 
 meetings.</p>
<?rh-cbt_start condition="Help" ?><p style="font-weight: bold;">&#160;</p><?rh-cbt_end ?>
<?rh-cbt_start condition="Help" ?><p style="font-weight: bold;">Visible 
 for</p><?rh-cbt_end ?>
<?rh-cbt_start condition="Help" ?><p>Click the <span style="font-weight: bold;">Visible 
 for</span> field to display a list that allows you to choose whether the 
 follow-up will be visible to all users, visible only to the user defined 
 as owner, or visible to all users in one of the user groups that the owner 
 belongs to.</p><?rh-cbt_end ?>
<?rh-cbt_start condition="Help" ?><p class="Note"><?rh-cbt_start condition="Help" ?><span 
 style="font-weight: bold;">Note:</span><?rh-cbt_end ?> If you specify 
 that an entry will only be visible to the owner, it will appear in italics 
 on the <?rh-cbt_start condition="Help" ?><span style="font-weight: bold;">Activities</span><?rh-cbt_end ?> 
 section tab.</p><?rh-cbt_end ?>
<?rh-cbt_start condition="Help" ?><p>This option is only available if <?rh-cbt_start condition="Help" ?><span 
 style="font-weight: bold;">Confidential activities</span><?rh-cbt_end ?>
