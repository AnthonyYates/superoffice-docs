# AddLinks

<ol style="list-style: decimal;">
<li class="p">Click the <span style="font-weight: bold;">Links</span> 
 tab.</li>
<li class="p">Click <span style="font-weight: bold;">Edit</span> to 
 switch to edit mode.</li>
<li class="p">Click <span style="font-weight: bold;">Add</span>. The 
 <span style="font-weight: bold;">Add link</span> dialog opens.</li>
<li class="p">Select the link type you want to add. The options are 
 <span style="font-weight: bold;">URL</span>, <span style="font-weight: bold;">Project</span>, 
 <span style="font-weight: bold;">Sale</span>, <span style="font-weight: bold;">Document</span> 
 and <span style="font-weight: bold;">Follow-up</span>.</li>
<p><img src="StandardCRM/Bitmaps/Chap08_project/86_chap8_add-link.bmp" 
 alt="" style="border: none;" border="0" /></p>
<li class="p">In the next field, select what you want to create a link 
 to. Use the <a href="StandardCRM/chap01/Using_the_history_list.htm">history 
 list</a> or start typing to search. The name of this field changes 
 according to the link type you selected above. If, for example, you 
 selected <span style="font-weight: bold;">URL</span> in the <span 
 style="font-weight: bold;">Type</span> field, here you enter the <span 
 style="font-weight: bold;">URL</span> address of the page you want 
 to display.</li>
<li class="p">Enter a brief description of the link in the <span style="font-weight: bold;">Description</span> 
 field. If you enter a description, this will be used as the link text.</li>
<li class="p">Click <span style="font-weight: bold;">OK</span> to save 
 the link.</li>
<li class="p">Click <span style="font-weight: bold;">Save</span> in 
 the <span style="font-weight: bold;">Links</span> tab.</li>
</ol>
<p>A dot on the <span style="font-weight: bold;">Link</span> tab header 
 indicates that a link has been added to the sale.</p>
<p>&#160;</p>
<p class="Prosedyre">Open links</p>
<p>You can open links by clicking on them in the <span style="font-weight: bold;">Links</span> 
 tab. URLs/websites open in a new window.</p>
<p>&#160;</p>
<p class="Prosedyre">Remove links</p>
<p><span><nobr>If you want to remove a</nobr> link, go to the </span><span 
 style="font-weight: bold;">Links</span><span> tab, click </span><span 
 style="font-weight: bold;">Edit</span><span>, click the delete button 
 next to the link. The link will be deleted from the tab.</span></p>
