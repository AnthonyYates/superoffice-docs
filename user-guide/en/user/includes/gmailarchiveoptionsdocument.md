# gmailarchiveoptionsdocument

<ul>
<li><span style="font-weight: bold;">Subject</span>: The subject is 
 automatically inserted in this field. However, you can edit the text.</li>
<li><span style="font-weight: bold;">Company</span>: Search for a company 
 by starting to type the company name or select a company in the list.</li>
<li><span style="font-weight: bold;">Contact</span>: Select a contact 
 in the list. You can also search by starting to type the contact name.</li>
<li><span style="font-weight: bold;">Sale</span>: If relevant, link 
 the document to a sale.</li>
<li><span style="font-weight: bold;">Project</span>: If relevant, link 
 the document to a project.</li>
<li><span style="font-weight: bold;">Date</span>: Today's date is automatically 
 inserted in this field. Click the field to select another date.</li>
<li><span style="font-weight: bold;">Type</span>: Select one of the 
 available document templates.</li>
<li><span style="font-weight: bold;">Visible for</span>: Select whether 
 the document will be visible to all users, private visible only to 
 the owner) or visible to all users in a user group to which the owner 
 belongs.</li>
<li><span style="font-weight: bold;">Mark as completed</span>: This 
 option is checked by default, but you can uncheck it if necessary.</li>
</ul>
