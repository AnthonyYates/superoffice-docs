<p class="Prosedyre">Add recipients from selections and projects</p>
<p><span>You can add recipients from selections (from both <?rh-udv_start 
 name="SO_User" ?>SuperOffice CRM<?rh-udv_end ?> and <?rh-udv_start 
 name="ProdName_CS" ?>SuperOffice Service<?rh-udv_end ?>) and projects 
 (from <?rh-udv_start name="SO_User" ?>SuperOffice CRM<?rh-udv_end ?>). 
 </span></p>
<p>To do this: </p>
<ol style="list-style: decimal;">
<li>Under <span class="UI-text">Add recipients</span>, click <span 
 class="UI-text">Selection</span> or <span class="UI-text">Project</span>.</li>
<li>Click in the field below and start typing the name of the selection 
 or project you want to add. As you type, a dynamic list displays the 
 matches.</li>
<li>Click a selection or a project in the list.</li>
<li>Click <img src="Mailing/Bitmaps/btn_Add.png" alt="" style="border: none;" 
 border="0" />. The selection or project is added to 
 the <span class="UI-text">Recipient sources</span> list, and the <span 
 class="UI-text">Recipients</span> list is updated.</li>
<li>Repeat steps 1 through 4 to add more selections or projects.</li>
</ol>
<p class="Note"><span style="font-weight: bold;">Note:</span> If you selected 
 a selection under <span class="UI-text">Archive mailing</span> in <span 
 class="UI-text">Setup</span> (step 1), this selection is listed under 
 <span class="UI-text">Recipient sources</span>. You can delete this selection 
 if necessary.</p>
<p class="Prosedyre">Add contacts from SuperOffice</p>
<p>You can add individual contacts from SuperOffice. To do this:</p>
<ol style="list-style: decimal;">
<li>Under <span class="UI-text">Add recipients</span>, click <span 
 class="UI-text">Contact</span>.</li>
<li>Click in the field below and start typing the name of the contact 
 you want to add, in the field below. You can also enter a company 
 name to display all contacts for a company.</li>
<li>Select the contact you want from the list that appears.</li>
<li>Click <img src="Mailing/Bitmaps/btn_Add.png" alt="" style="border: none;" 
 border="0" />. The contact is added to the <span class="UI-text">Recipient 
 sources</span> list, and the <span class="UI-text">Recipients</span> 
 list is updated.</li>
<li>Repeat steps 2-5 to add more contacts. Additional contacts will 
 be added to the same row in the <span class="UI-text">Recipient sources</span> 
 list, and the number of contacts will display in the <span class="UI-text">Name</span> 
 column.</li>
</ol>
