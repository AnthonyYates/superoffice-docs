# Crosstable viewer

<p>To view a cross table report (.Dcube file), you need to install the 
 <span style="font-weight: bold;">CrossTable viewer</span>.</p>
<p>Do as follows:</p>
<ol style="list-style: lower-roman;">
<li class="p">Click <img src="StandardCRM/bitmaps/globalmenu_mainmenu_small.png" 
 alt="" style="border: none;" border="0" /> 
 (main menu) and select <span style="font-weight: bold;">Download &gt; 
 Download CrossTable viewer</span>.</li>
<li class="p">Click the downloaded file (<span style="font-weight: bold;">SoCrossTableInstaller.exe</span>) 
 to run it.</li>
<li class="p">If a security warning is displayed, confirm that you 
 want to run the file. The installation wizard then opens.</li>
<li class="p">Follow the instructions in the wizard to install the 
 program.</li>
</ol>
<p>You only need to do this once per computer.</p>
