# SERVICE_getnextinqueue

<p class="Note"><span class="UI-text">Get next request in queue</span>: 
 If the queue system is set up for your company, you can click this option 
 to get the next unassigned request in the queue for the category you belong 
 to. To set up this feature, see <a href="Service/topics/Get_next_request_in_the_queue.html">How 
 to set up Get next request in queue</a>.</p>
