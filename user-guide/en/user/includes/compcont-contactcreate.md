# CompCont_ContactCreate

<ol start="2" style="list-style: decimal;">
<li class="p">In the top section of the tab, you enter the following 
 information: 
<ul>
<li class="p">Contact's first name and/or last name. Mandatory: 
 You must enter either a first name or a last name.</li>
<li class="p">Salutation (<b><i>Mr/Ms</i></b>)</li>
<li class="p">Company. Select a company in the list or start typing 
 a company name to search. If the contact should not be connected 
 to a company, select <span style="font-weight: bold;">(No Selection)</span>.</li>
<li class="p">Contact image. You can add a photo by dragging the 
 image file from Windows Explorer and dropping it on the image 
 field at the top left of the tab. </li>
</ul>
<p class="Note">You can also add a picture by right-clicking the picture 
 field at the top left of the tab. Click the link and browse to the 
 picture you require and click <b>Open</b> to add the picture.</p></li>
<li class="p">Enter contact information in the <span style="font-weight: bold;">Contact</span> 
 tab. See <a href="StandardCRM/person/contactcard_tab_contact.htm">The 
 Contact tab</a>.</li>
<li class="p">Go to the <span style="font-weight: bold;">Details</span> 
 tab. Here you can enter additional information for the contact. See 
 <a href="StandardCRM/person/contactcard_tab_details.htm">The Details 
 tab</a>.</li>
<li class="p">Go to the <b>More</b> tab. This tab contains user-defined 
 fields for the contact. See <a href="StandardCRM/person/contactcard_tab_more.htm">The 
 More tab</a>.</li>
<li class="p">Go to the <b>Interests</b> tab. Here you can set interests 
 and e-marketing/subscription options for the contact. See <a href="StandardCRM/person/contactcard_tab_interests.htm">The 
 Interests tab</a>.</li>
<li class="p">Go to the <span style="font-weight: bold;">Note</span> 
 tab. Here you can enter information of any kind. See <a href="StandardCRM/person/contactcard_tab_note.htm">The 
 Note tab</a>.</li>
<li class="p">Go to the <span style="font-weight: bold;">www</span> 
 tab. See <a href="StandardCRM/person/contactcard_tab_www.htm">The 
 www tab</a>.</li>
<li class="p" style="list-style: decimal;">Click <span style="font-weight: bold;">Legal 
 basis</span> at the bottom of the contact card. The <span style="font-weight: bold;">Privacy</span> 
 dialog opens. Review and edit the consent information as necessary. 
 See <a href="StandardCRM/chap03/Edit_legalbase.htm">Edit privacy settings 
 for a contact</a>. Click <span style="font-weight: bold;">Close</span> 
 when you are done.</li>
<li class="p">Check <b>Former employee</b> (at the bottom of the contact 
 card) if this contact has stopped working for the company. A contact 
 who is recorded as a former employee is not normally displayed in 
 the lists of contacts or project members.</li>
<li class="p">When you have entered the necessary information, click 
 the <b>Save</b> button. The contact information is then saved in the 
 database and displayed in the contact card. If you have linked the 
 contact to a company, the contact will be displayed in the <span style="font-weight: bold;">Contacts</span> 
 tab in the <span style="font-weight: bold;">Company</span> screen.</li>
<li class="p">Go to the <span style="font-weight: bold;">Relations</span> 
 section tab. Here you can add relations between this contact and another 
 contact or company. See <a href="StandardCRM/person/contact_secttab_relations_add.htm">Add 
 relations</a>.</li>
</ol>
