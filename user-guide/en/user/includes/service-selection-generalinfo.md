# SERVICE_selection_generalinfo

<ul style="list-style-type: disc;">
<li><span class="UI-text">Name</span>: Enter a descriptive name for 
 the selection.</li>
<li><span class="UI-text">ID string</span>: Here you can enter a unique 
 identifier. In a web-based customer centre, you can refer to this 
 ID string to make the selection available to customers.</li>
<li><span class="UI-text">Folder</span>: If you have organised the 
 selections into a folder structure (see <a href="Service/topics/selections.editSelectionFolder.html">Administrate 
 folders of selections</a>), click <img src="Bitmaps/btn_dropdown.png" 
 alt="" align="middle" border="0" /> 
 and select a folder.</li>
<li><span class="UI-text">Access</span>: In this list box, you select 
 who is to have access to read and edit the selection.</li>
<li><span class="UI-text">Show in left menu</span>: Check here if you 
 want the name of the selection to appear under <span class="UI-text">Selections</span> 
 in the Navigator.</li>
<li><span class="UI-text">Field for parameter constraint</span>: This 
 field can be used to enter a key or a filter in order to display a 
 selection as a web panel.</li>
</ul>
