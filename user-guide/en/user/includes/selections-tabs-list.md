# selections_tabs_list

<ul>
<li><a href="StandardCRM/chap04/The_List_of_Members.htm">The list of 
 members</a>: All selection types except <span style="font-weight: bold;">Company/contact</span> 
 will have two tabs containing selection data. Selection members can 
 be projects, sales, documents, requests, follow-ups, form submissions, 
 chat conversations, mailings or products.</li>
<li><a href="StandardCRM/chap04/selection_tab_Company_contact.htm">Company/contact</a>: 
 The <span style="font-weight: bold;">Company/contact</span> tab contains 
 companies and contacts linked to list of members. Example: All contacts 
 or companies linked to the sales in a selection.</li>
<li><a href="StandardCRM/chap04/selection_tab_Criteria.htm">Criteria</a>: 
 Here you can edit the criteria for a dynamic selection. </li>
<li><a href="StandardCRM/chap04/selection_tab_Details.htm">Details</a>: 
 Here you can edit general information for a selection.</li>
<li><a href="StandardCRM/chap04/Display_selections_as_charts.htm">Charts</a>: 
 Here you can view the selection data in a chart. Charts of selections 
 can also be used in <a href="StandardCRM/Dashboard/Dashboard_Create.htm">dashboards</a>.</li>
<li><a href="StandardCRM/chap04/Mailings_section_tab.htm">Mailings</a>: 
 Here you can create a mailing to contacts in a selection. You can 
 also see the mailings that have been archived on this selection.</li>
</ul>
