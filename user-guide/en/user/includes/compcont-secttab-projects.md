# CompCont_SectTab_Projects

 screen in <?rh-udv_start name="SO_User" ?>SuperOffice CRM<?rh-udv_end ?> 
 which contains information about the active project. </p>

 tab contain the following information:</p>
<ul>
<li class="p">Check mark indicating if the project has been completed.</li>
<li class="p">The name of the project.</li>
<li class="p">What type of project it is.</li>
<li class="p">The project's status.</li>
<li class="p">The project number.</li>
<li class="p">The project description (if any). Hold the mouse-pointer 
 over the paper-clip icon ( <img src="StandardCRM/bitmaps/Binders.bmp" 
 alt="" style="border: none;" border="0" /> 
 ) to display the description.</li>
<li class="p">Any link to a web page the project is connected to â€“ 
 a URL address to click on to go to the relevant web page.</li>
<li class="p">User ID for the person who created the project.</li>
</ul>
