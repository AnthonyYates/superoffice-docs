# Service_KB_FolderAdmin_create

<p class="Prosedyre">Create folders</p>
<p>To create a new folder:</p>
<ol style="list-style: decimal;">
<li>Go to the folder you want to create a subfolder under.</li>
<li>Click the <span class="UI-text">New folder</span> button at the 
 top of the screen.</li>
<li>The <span class="UI-text">Folder</span> field automatically shows 
 the name of the parent folder you chose. You can select a different 
 folder here.</li>
<li>Enter a descriptive name for the folder in the <span class="UI-text">Name</span> 
 field.</li>
<li>In the <span class="UI-text">Description</span> field, enter a 
 brief description of what the folder contains.</li>
<li>Click <span class="UI-text">OK</span>. The new folder is created.</li>
</ol>
