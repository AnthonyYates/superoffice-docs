# Selections_AddMembers_Other

<p>In the section tab containing the selection members, do as follows:</p>
<ol style="list-style: decimal;">
<li class="p">Click the <span style="font-weight: bold;">Add Members</span> 
 button. The <span style="font-weight: bold;">Add Members</span> dialog 
 opens.</li>
<p class="Note"><span style="font-weight: bold;">Note:</span> The <span 
 style="font-weight: bold;">Add Members</span> dialog is also displayed 
 automatically when you save a new selection. See <a href="StandardCRM/chap04/Creating_static_selections.htm">Create 
 static selections</a>.</p>
<?rh-cbt_start condition="Print" ?><p><img src="StandardCRM/Bitmaps/Chap09_selection/chap9_add-members_static_comp_other.bmp" 
 alt="" style="border: none;" 
 border="0" /></p><?rh-cbt_end ?>
<li class="p">Click the <span style="font-weight: bold;">Add</span> 
 button.&#160;A new line is added to the criteria list and a list of 
 fields to choose from is displayed.</li>
<li class="p">Select search criteria in the different fields by clicking 
 the arrow ( <img src="StandardCRM/bitmaps/Dialogboksknapper_pil ned.bmp" 
 alt="" style="border: none;" border="0" /> ) and 
 select from the list box that appears. The criteria displayed here 
 depend on the type of selection you have created. The search criteria 
 for selections work in the same way as for standard searches.<?rh-cbt_start condition="Help" ?> 
 See <?rh-cbt_end ?><?rh-cbt_start condition="Help" ?><a href="StandardCRM/chap01/Using_search_criteria.htm">Use 
 search criteria</a><?rh-cbt_end ?>.</li>
<li class="p">Repeat points 2 and 3 until you have specified all the 
 search criteria you require.</li>
<li class="p">Click <span style="font-weight: bold;">OK</span>. A dialog 
 box displays the number of selection members that have been added 
 to the selection.</li>
</ol>
