# Search_List_field

<p>If the field you are searching in is a field containing lists, e.g. 
 the <span style="font-weight: bold;">Business</span> field in the <span 
 style="font-weight: bold;">Company</span> screen, a list box appears 
 with the options <span style="font-weight: bold;">Is one of</span> and 
 <span style="font-weight: bold;">Is not one of</span>.</p>
<p>Click the <span style="font-weight: bold;">Select items</span> field 
 to display the list of values.</p>
<p>If the list is long, you can type text in the <span style="font-weight: bold;">Search 
 criteria</span> fields to search for values.</p>
<p>Click outside the list to close it.</p>
