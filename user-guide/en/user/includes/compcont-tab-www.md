# CompCont_tab_www

<p style="margin-left: 3px;">In the <span style="font-weight: bold;">www</span> 
 tab, you can specify a shortcut to a URL address, in order to get information 
 from other applications via a URL. The URL address can be specified using 
 a template variable. The <span style="font-weight: bold;">www</span> tab 
 can be displayed in several places in <?rh-udv_start name="SO_User" ?>SuperOffice CRM<?rh-udv_end ?>, 
 as defined in SuperOffice Settings and maintenance. If you click a shortcut, 
 the linked URL address opens.</p>
<p class="Note"><span style="font-weight: bold;">Tip:</span> In the <span 
 style="font-weight: bold;">Labels</span> tab in the <span style="font-weight: bold;">Options</span> 
 window in SuperOffice Settings and maintenance, you can specify a different 
 name for this tab than <span style="font-weight: bold;">www</span>. You 
 can also specify a different location in <span style="font-weight: bold;">GUI 
 - Web panel</span> in SuperOffice Settings and maintenance.</p>
<p>&#160;</p>
<p>Here you can click <span style="font-weight: bold;">Open in separate 
 window</span> to view the web page in a separate browser window.</p>
