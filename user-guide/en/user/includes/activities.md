# Activities

<p><img src="StandardCRM/Bitmaps/Chap04_activities/30_chap4_activities-detail.bmp" 
 alt="" style="border: none;" border="0" /></p>
<p>&#160;</p>
<p>The <b>Activities</b> section tab is available in the <b>Company</b> 
 screen, the <span style="font-weight: bold;">Contact</span> screen, the 
 <b>Project</b> screen, the <b>Diary</b> screen and the <b>Sale</b> screen. 
 You create activities using the <img src="StandardCRM/bitmaps/globalmenu_new.png" 
 alt="" style="border: none;" border="0" />&#160;<span 
 style="font-weight: bold;">New</span> menu in the top bar, or by double-clicking 
 in the section tab.</p>
<ul style="list-style: disc;">
<li>The <b>Activities</b> section tab in the <b>Company</b> screen 
 displays activities linked to the company in question. <span>See <a 
 href="StandardCRM/chap03/Activities_Contact.htm">Activities (Company)</a>.</span></li>
<li>The <b>Activities</b> section tab in the <b>Contact</b> screen 
 displays activities linked to the contact in question. <span>See <a 
 href="StandardCRM/person/contact_secttab_activities.htm">Activities 
 (Contact)</a>.</span></li>
<li>The <b>Activities</b> section tab in the <b>Project</b> and <span 
 style="font-weight: bold;">Sale</span> screens displays activities 
 linked to the project or sale in question. <span>See <a href="StandardCRM/chap05/Activities_Project.htm">Activities 
 (Project)</a> and <a href="StandardCRM/sale/activities_sale.htm">Activities 
 section tab (Sale)</a>.</span></li>
<li>The <span style="font-weight: bold;">Activities</span> section 
 tab in the <b>Diary</b> screen displays all your uncompleted activities. 
 <span>See <a href="StandardCRM/chap06/activities_list.htm">Activities 
 section tab (Diary)</a>.</span></li>
<li><p>Activities are also displayed in the different views in the 
 <span style="font-weight: bold;">Diary</span> screen. See <a href="StandardCRM/chap06/Tabs_in_the_Diary_screen.htm">Tabs 
 in the Diary screen</a>.</p></li>
</ul>

 is a collective term for appointments, phone calls and tasks. In addition, 
 mailings, form submissions and chat records are displayed in the <span 
 style="font-weight: bold;">Activities</span> section tab.</p>

<p>&#160;</p>
<p>The activities list columns contain the following information:</p>
<ul style="list-style: disc;">
<li><img src="StandardCRM/bitmaps/Utfoert.bmp" alt="" style="border: none;" 
 border="0" />: The first column in the activities list specifies 
 the <b>Completed</b> status of an activity. A tick means that the 
 activity is completed, while an empty box means that it is not completed.</li>
<p class="Note"><b style="font-weight: bold;">Tip:</b> To change the 
 <b>Completed</b> status of an activity simply click the box.</p>
<li><span style="letter-spacing: -.25pt;">The </span><span style="letter-spacing: -.25pt; 
 font-weight: bold;">Category</span><span style="letter-spacing: -.25pt;"> 
 column contains icons that indicate the type of activity, for example, 
 a telephone for a call. These icons reflect the text in the <b>Type</b> 
 column.</span></li>
<li>The other columns in the activities list show various information 
 about the activity. If not all the text in a field is visible, hold 
 the mouse pointer over it to display it in a tooltip.</li>
<p class="Note"><b>Tip:</b> To display more information in the <b>Activities</b> 
 section tab, you can configure the columns to show the information 
 that is important to you. <span>See <a href="StandardCRM/chap01/Changing_the_column_view.htm">Configure 
 columns</a>.</span></p>
</ul>
<p>&#160;</p>
<p>The activities displayed depend on which of the following checkboxes 
 you have selected at the bottom of the section tab:</p>
<ul style="list-style: disc;">
<li>Follow-ups (appointments, tasks and phone calls)</li>
<li>Documents (e-mail, letters, and other documents)</li>
<li>Mailings (bulk mailings by merged document, e-mail or SMS)</li>
<li>Other (chat records, form submissions)</li>
</ul>
<p>You can show all activities, or just one type.</p>
