# Activate_deactivate_search_criteria

<p>When you open the <span style="font-weight: bold;">Find</span> screen, 
 it contains default search criteria or the search criteria you specified 
 the last time you used the dialog. If you want to use different search 
 criteria than those shown, you have the following options:</p>
<ul style="list-style: disc;">
<li>you can delete them by clicking <img src="StandardCRM/bitmaps/criteria_remove.png" 
 alt="" style="border: none;" 
 border="0" /></li>
<li>you can deactivate one or more search criteria by leaving the value 
 field empty. A deactivated search criterion remains in the criteria 
 list and can be simply reactivated when needed. </li>
<li>you can deactivate all the search criteria by clicking <span style="font-weight: bold;">Clear 
 all</span>.</li>
</ul>
