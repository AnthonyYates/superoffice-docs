# Service_Mailing_SMTP_fields

<ul>
<li><span class="UI-text">Host</span>: The IP address or host name of the e-mail server used to send e-mail. The server must be accessible from the SMTP port that the SuperOffice applications run on.</li>
<li><span class="UI-text">Port</span>: Here you specify which SMTP port the SuperOffice applications run on. The default port is 25.</li>
<li><span class="UI-text">User name</span> / <span class="UI-text">Password</span>: Enter the user name and password for the server.</li>
<li><span class="UI-text">Use SSL</span>: Check this option to use secure socket layer when logging on.</li>
<li><span class="UI-text">Check SMTP connection</span>: If you check here, the system will try to send an e-mail to an address at <?rh-udv_start name="SO_company_name" ?>SuperOffice AS<?rh-udv_end ?>. The purpose of this is to establish if the SMTP server allows you to send out e-mail from your own domain. If you do not check here, no check will be made.</li>
</ul>
