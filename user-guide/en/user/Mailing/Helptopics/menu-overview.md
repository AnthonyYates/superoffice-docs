# Menu overview

Below is an overview of the various functions in the **Content** step.

## Side-panel

The side-panel contains the following main sections:

<table>

<tbody>
<tr>
<td><p>**Icon**</p></td>
<td><p>**Name**</p></td>
<td><p>**Function**</p></td>
</tr>
<tr>
<td>![icon](../media/editor-sidepanel-btn-content.png)</td>
<td>Content</td>
<td>This section contains various content types, such as columns, text sections, images, menus/buttons and SoMe links. Drag and drop content from this section to the message. Click the content to edit it in the side panel. See [Edit content](Edit-a-paragraph.md).</td>
</tr>
<tr>
<td>![icon](../media/editor-sidepanel-btn-blocks.png)</td>
<td>Blocks</td>
<td><p>This section contains saved content blocks such as headers and footers. Drag and drop content from this section to the message.</p>
<p>To save a content block, click a row and select **Save block** (![icon](../media/NEWbtn-SaveBlock.png)).</p></td>
</tr>
<tr>
<td>![icon](../media/editor-sidepanel-btn-body.png)</td>
<td>Body</td>
<td>In this section you can edit the global properties and layout for the message. > [!NOTE]
> If you have defined specific properties for individual sections, these are not overridden by global properties.</td>
</tr>
<tr>
<td>![icon](../media/editor-sidepanel-btn-images.png)</td>
<td>Images</td>
<td>In this section you can find millions of images to use in your mailing. Drag and drop content from this section to the message. See [Insert images in the message](Insert-images-in-the-message.md).<br />
> [!NOTE]
> If the **Images** (![icon](../media/editor-sidepanel-btn-images-small.png)) section is not visible, this preference may have been disabled due to organisational privacy rules.</td>
</tr>
</tbody>
</table>

See also [Add content](Add-content.md).

## Context menu

When you click a section in a message, you get access to various functions for the section.

<table>

<tbody>
<tr>
<td><p>**Icon**</p></td>

<tr>
<td><p>![icon](../media/NEWbtn-AddRow.png)</p></td>
<td><p>Add row</p></td>

</tr>
<tr>
<td><p>![icon](../media/NEWbtn-editor-move.png)</p></td>
