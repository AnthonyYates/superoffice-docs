# The Inbox

The inbox contains the following options:

## Toolbar at the top

<table class="SimpleGridBlack" data-cellspacing="0px">

<tbody>
<tr>
<td>**New**</p></td>
<td>Click here to create a new message. See [Compose an e-mail](Writing-e-mail.md).</p></td>
</tr>
<tr>
<td>**Delete**</p></td>
<td>Delete: Click here to delete the selected messages.</p></td>
</tr>
<tr>
<td>![icon](../../media/globalmenu-search.png)</p></td>
<td>Search: Enter text in this field to start searching in the open folder.</p></td>
</tr>
<tr>
<td>![icon](../media/web-email-resetsearch.png)</p></td>
<td>Click this button to remove the search result and return to the folders original content.</p></td>
</tr>
<tr>
<td>INBOX</p></td>
<td>Click here to select a folder. The default folder is Inbox.</p></td>
</tr>
<tr>
<td>![icon](media/loc/mail-Sort.png)</p></td>
<td>By default the messages are sorted by date, with the newest messages at the top of the list. You can also sort by sender and subject in descending or ascending order.</p></td>
</tr>
</tbody>
</table>

## Toolbar at the bottom

<table class="SimpleGridBlack" data-cellspacing="0px">

<tbody>
<tr>

</tr>
<tr>
<td>![icon](../media/Epost-oppdater.bmp)</p></td>
<td>Refresh: Click here to refresh the inbox.</p></td>
</tr>
</tbody>
</table>
