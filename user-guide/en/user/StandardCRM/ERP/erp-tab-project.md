# ERP (project)

If your organisation has a licence for SuperOffice Integration Server, the **ERP** section tab is displayed in the **Project** screen. Here you can create a connection between SuperOffice projects and projects in the ERP system.

If a connection has already been created, you can click **Sync Now** to synchronise the data between SuperOffice CRM and the ERP system.
