# Upload documents

To quickly upload a document to SuperOffice CRM, do as follows:

1. Click **New** &gt; **Document** on the top bar. The **Document** dialog opens.

2. Drag and drop the document from Windows Explorer to the **Document** dialog.

3. (optional) Click ![icon](../media/followup_notcompleted_SMALL.png) in the top-right corner to set the document to **Completed** (![icon](../media/followup_completed_SMALL.png)).

4. Complete the fields with the required information. See [The Document dialog](Document_dialog.md) for an explanation of what to enter here.

5. Click the **Create** button to create the new document in SuperOffice CRM. The document is added to the **Activities** section tab.
