# Edit favourite reports

To edit a favourite report:

1. Open the **Reports** screen.  
    <!-- Fix reuse ID=a1 -->
    To open the **Reports** screen:
    * By clicking the **Reports** button in the Navigator.
2. Select the **Favourites** tab. The favourite reports are displayed in the report list.

> [!NOTE]
> This assumes that you have added some favourite reports (see [Add favourite reports](Adding-favourite-reports.md)).

1. Double-click the report you want to edit, or select it and click **Edit**. A dialog opens in which you can edit the report's details.
2. Edit the name of the report in the **Please enter a name for your report field**, as required.
3. Edit the text in the **Report description** field, as required.
4. Click **OK**.

> [!NOTE]
> The report is amended immediately and you cannot undo your changes.
