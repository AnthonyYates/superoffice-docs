# The Navigator

You move between the main sections of SuperOffice CRM using the Navigator, which is on the far left of the main screen.

> [!TIP]
> Clicking the word next to the button gives you access to various options, such as favourites list, history list and search. If you hold the mouse pointer over a record in any the lists, a <!-- Fix reuse ID=a1 --> containing additional data about the record.

![icon](../media/Chap02-GettingStarted/Navigator-preview.png)

> [!TIP]
> You can change the width of the navigator by clicking these buttons above the navigator: ![icon](../../media/panel-left-hide.png) ![icon](../../media/panel-left-show.png)

The navigator contains the following buttons:

<tbody>
<tr>
<td><p>![icon](../media/Navigator-Dashboard.bmp)</p></td>
<td><p>**Dashboard**: opens the **Dashboard** screen, containing reports and charts of your SuperOffice data. See [Dashboard](../Dashboard/Dashboards.md).</p></td>
</tr>
<tr>
<td><p>![icon](../media/Navigator-Kontakt.bmp)</p></td>
<td><p>**Company**: opens the **Company** screen, which is your window to the database containing all the companies you are in contact with. See [Companies](../chap03/Companies.md).</p></td>
</tr>
<tr>
<td><p>![icon](../media/Navigator-Contact.bmp)</p></td>
<td><p>**Contact**: opens the **Contact** screen, which is your window to the database containing all your contacts. See [Contact](../person/Contact-main.md).</p></td>
</tr>
<tr>
<td><p>![icon](../media/Navigator-Dagbok.bmp)</p></td>
<td><p>**Diary**: opens the **Diary** screen, which is a kind of advanced filofax. See the main [Diary](../chap06/Diary.md) topic.</p></td>
</tr>
<tr>
<td><p>![icon](../media/Navigator-Salg.bmp)</p></td>
<td><p>**Sale**: opens the **Sale** screen, containing information about all the sales. See the main [Sale](../sale/sale.md) topic.</p></td>
</tr>
<tr>
<td><p>![icon](../media/Navigator-Prosjekt.bmp)</p></td>
<td><p>**Project**: opens the **Project** screen, containing information about different projects. See [Project](../chap05/Project.md).</p></td>
</tr>

<td><p>![icon](../media/Navigator-Utvalg.bmp)</p></td>
<td><p>**Selection**: opens the **Selection** screen, where you can enter companies, contacts, projects, sales, documents and follow-ups that you work with frequently into separate selections. </tr>
<tr>
<td><p>![icon](../media/Navigator-E-post.bmp)</p></td>
