# Section tabs in the Diary screen

The **Diary** screen contains the following section tabs:

[Activities](activities-list.md): The **Activities** section tab gives you an overview of activities and acts as a to-do list or task list.

[Sales](Sale-card.md): The **Sales** section tab shows essential information about sales.
