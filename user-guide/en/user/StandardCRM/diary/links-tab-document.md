# The Links tab (Document)

This tab contains information about activities (documents and follow-ups) and sales that are linked to the document in question. If anything is registered in this tab, a dot is displayed next to **Links**. For more information on how to add links, see [Add links from the dialog for follow-ups or the Document dialog](Linking-documents-to-follow-ups.md).
