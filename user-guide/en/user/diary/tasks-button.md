# Task button

If you select the **View** tab in the **Diary** screen, the ![icon][img1] **Task** button is displayed.

The ![icon][img1] **Task** button allows the follow options:

* Select **Edit** to edit an existing view. Predefined views cannot be edited.
* Select **[New group view][1]** to create a new group view.
* Select **[Copy][1]** to make a copy of the group view you have opened.
* Select **[Delete group view][1]** to delete the group view you have opened.
* Select [Invite participants][2] if you want to link participants to a follow-up.
* Select **Help** to open this help file.
