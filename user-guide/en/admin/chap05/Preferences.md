# Preferences

In the **Preferences** screen you can set preferences for the whole system (globally), one database, one user group or an individual user (see [Levels for preference settings](Levels-for-preference-settings.md)). Preferences that can only be set on a global (system) level, can be found in the <a href="Global-preferences.md" style="font-weight: bold;">Global preferences</a> tab.

In addition, you can configure video meetings in the **[Video meetings](Video-meetings.md)** tab and document library in the **[Document library](Document-library.md)** tab.
