# Delete mailboxes

To delete a mailbox, do as follows:

1. Click the **Requests** button in the navigator and select the **E-mail** tab.
2. Select the **Mailboxes** tab.
3. Click the required mailbox. The properties screen appears.
4. Click the **Delete** button at the bottom of the screen. The mailbox is removed.
