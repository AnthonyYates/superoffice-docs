# SuperOffice and licences

To log in to SuperOffice CRM from a Windows or Web client you need the following licences:

* Login rights to SuperOffice CRM
* A client licence to Standard CRM

The following provides a summary of the various SuperOffice CRM licence types.

> [!NOTE]
> The licences which are displayed in the **Licences** screen depend on which licences you have purchased and activated. The **Licences** screen may contain more or fewer licences than described below.

## SYSTEM LICENCES

A "system" licence specifies which systems you have access to.

* **SuperOffice Server**: This licence is required in order to use SuperOffice CRM, SuperOffice Service and the other SuperOffice modules.
* **Online WebServices**: This licence is required in order to access SuperOffice CRM, SuperOffice Service and the other SuperOffice modules from a web client.
* **Area Management**: allows the definition of areas in Settings and maintenance (see Area Management).
* **Sales Intelligence**: allows SAINT status monitors (see [Status monitors][2]).
* **Exchange Synchronizer**: allows use of SuperOffice Synchronizer.
* **SuperOffice Expander Services**: allows functionality for anonymous and external users via SuperOffice Audience or third-party applications, among other things.
* **Customer Centre**: allows use of the customer centre module in SuperOffice Service.
* **Sales Quote Management**: allows use of quote module in Sale.

## SITE LICENCES

There are also so-called site licences. A "site" in SuperOffice terms corresponds to a database or a satellite database. A site licence specifies the following:

* **Satellite**: the number of satellite databases your organisation can define (see Satellite).
* **Reporter Studio**: the maximum number of sites which can use SuperOffice Reporter Studio.
* **Audience**: The maximum number of sites which can use SuperOffice Audience.
