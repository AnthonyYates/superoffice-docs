# User licences

On the **User licences** tab in the **Users** screen, you can get an overview of the licence usage and distribution.

* Click a licence in the **User licences** list to display a list of the users who are currently using this licence.
* Hold the mouse-pointer over a licence name to see detailed information about the licence.
