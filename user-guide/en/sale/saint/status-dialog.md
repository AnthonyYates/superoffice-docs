# The View statuses dialog

[!include[Requirement](includes/req-saint.md)]

The **View statuses** dialog is displayed when you click a status image for a company card, contact card or project card. The dialog contains the following information:

**\[Name\]:**

The name of the status. This is defined in SuperOffice Settings and maintenance.

**\[Description\]:**

A description of the status. This is defined in SuperOffice Settings and maintenance.

**Other statuses:**

A link to any other statuses relating to the current company, contact or project. This field is only displayed if there are several statuses for the record in question.

**Create follow-up:**

Click here if you want to create a new follow-up for the company, contact or project. The type of follow-up created by default is defined in SuperOffice Settings and maintenance. [!include[See Appointment dialog](../../includes/see-apt-dialog.md)]

**Add to selection:**

When you click this button, the **Add to selection** dialog opens and here you can add the current company or contact to a static selection of your choice.

> [!NOTE]
> This function can only be used for static selections. For more information about how to use statuses in dynamic selections, see [Use status monitors in dynamic selections][1]

**Close:**

Click here to close the status dialog.
