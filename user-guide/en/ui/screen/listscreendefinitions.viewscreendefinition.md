# Screen

[!include[Requirement](../../includes/req-expander-services.md)]

Here you can view the details of the selected screen.

* **Edit screen**: Click here to edit the screen. See [Screen properties][1].

* **Edit language**: Click here to edit screen texts for the available languages. See [Edit language][2].

* **New element**: Click here to add a new element to the screen. See [Element properties][3].

* **New script**: Click here to add a new script. See [Edit button][4].

* **New macro**: Click here to add a new macro. See [Create a macro][5].

* **Show screen**: Click here to show the screen.

* **Design mode**: Click here to show the screen in design mode (see [Design screen][6]). The following tabs are displayed:

  * **Elements**: The elements in the screen.
  * **Buttons**: Lists the buttons which are connected to actions in the screen.
