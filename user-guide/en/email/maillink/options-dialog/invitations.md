# Invitations

These options appear in the **Invitations** pane of the **SuperOffice CRM** options**** dialog (see [SuperOffice CRM options dialog][1]). Here you can define how invitations are handled.

Incoming invitations

When you respond to an invitation in Outlook (accepted or tentative), you can also create or update an appointment in SuperOffice.

Select this option: **Add / update appointment in SuperOffice when responding to invitations in Outlook**
